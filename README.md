# Basic procedural City
A basic procedural 3D city built with Babylonjs


## Installation
```
$ git clone https://github.com/Coac/basic-procedural-city
$ cd basic-procedural-city
$ npm install
```

## Development
```
$ npm run dev
$ open http://localhost:8080/
```
Running the webpack-dev-server.
